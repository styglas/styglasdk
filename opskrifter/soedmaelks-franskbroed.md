## Konfiguration

| | |
|---|---|
| Program | 1 (ca. 3 timer) — Alpina brødmaskine |
| Forsinket start | Nej |

**Må kun bages i brødstørrelse 500g — ellers bliver det for stort.**

## Flydende ingredienser

| Ingrediens | 500g |
|-----------|------|
| Mælk | 180ml |
| Rapsolie | 20g |
| Olivenolie | 10g |

## Tørre ingredienser

| Ingrediens | 500g |
|-----------|------|
| Hvedemel | 300g |
| Sukker | 1 tsk. |
| Salt | 1 tsk. |
| Gær | 1½ tsk. |
